# genius_ino
A versão para Arduino do jogo Genius!

# Itens necessários
para este projeto serão necessários:

*Hardware*
- Arduino/Genuino UNO (ou derivada)
- Cabo USB
- Protoboard
- 4 LEDs de cores diferentes
- 4 Botões
- Jumper Cables
- Buzzer

*Software*
- [IDE Arduino](https://www.arduino.cc/en/Main/Software)
